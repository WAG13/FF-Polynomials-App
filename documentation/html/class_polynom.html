Detailed Description

class Polynom that describes polynomial in a ring GF(p)[X]

polynomials over field GF(p) (p is prime) coefficient by (mod p) power of the polynomial - n Every object consists of terms (PolyTerm) Every structure "PolyTerm" has fields "key" and "next" key - coefficient of term next - pointer to next object of structure "PolyTerm"
